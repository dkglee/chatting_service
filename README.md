## 채팅 서버 프로그램

- **프로젝트 이름** : 게임에서 사용되는 실시간 채팅 서버

- **개발 기간** : 03.06 ~ 03.20

## 설치 및 사용 방법
** install program **
- `git clone https://github.com/dkglee/chatting_service.git`
  
** start server program ** : 루프백 주소(`127.0.0.1`)로 `8080` 포트로 서버가 열리게 된다.
- `./chatting_service`
  
** 클라이언트로 서버 연결 **
- `nc 127.0.0.1 8080`

** 연결 후 `username` 입력 **
